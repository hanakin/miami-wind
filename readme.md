# Miami Wind

This is a color scheme currently just for vscode/cursor with plans to expand it to support anything. The concept is based on the color scheme Miami Nights for [MonkeyType](https://monkeytype.com/), Combined with the base UI greyscale themeing from [Catppuccin](https://catppuccin.com/). The MonkeyType colors have been expanded and replaced with coresponding and chromatically similar colors from [Tailwind CSS](https://tailwindcss.com/)
